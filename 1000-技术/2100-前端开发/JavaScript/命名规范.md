# 命名规范

1. 项目名小写加横杠 : `my-project`
2. 类名大驼峰：`MyCalss`
3. 变量名小驼峰：`myVariable`
4. 常量名大写加下划线 `MY_CONST` :

## 所有命名规则必须遵循以下规则

- 名称只能由字母、数字、下划线、$符号组成
- 不能以数字开头
- 名称不能使用语言中的关键字
- 坚决不允许出现中文及拼音命名
