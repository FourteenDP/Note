# 通过 Vue-cli 创建 UniApp-Vue2 项目

## 安装 Vue-cli

```` shell
npm install -g @vue/cli@4
````

## 创建 UniApp

```shell
vue create -p dcloudio/uni-preset-vue uniapp-vue2-template
```

- 选择模板，看个人需求选择这里选择 **默认模板**
