# 跨域

跨域是指浏览器的同源策略限制了不同源的文档或脚本如何来进行通信。它是浏览器最核心也是最基本的安全功能，用于隔离潜在恶意的文档，减少可能被攻击的媒介。

## 同源策略

| 说明           | 是否允许通讯 | URL 示例 |
| ------------ | ------ | ---- |
| 同一域名，不同文件或路径 | 允许     |   <http://www.example.com/one.html> <br> <http://www.example.com/two.html> |
| 同一域名，不同端口    | 不允许    |
| 同一域名，不同协议    | 不允许    |
| 域名和域名对应 ip   | 不允许    |
| 域相同，子域不同     | 不允许    |
| 不同域名         | 不允许    |
