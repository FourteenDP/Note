# CORS

CORS 是 Cross-Origin Resource Sharing 的缩写，它是一种跨域资源共享的方式。CORS 的原理是利用浏览器的同源策略，只要服务器设置了正确的 CORS 响应头，浏览器就会允许跨域请求。
