# 让整个网站或盒子变成灰色

## 用 CSS 滤镜使整站或盒子变成灰色

```css
filter: grayscale(100%);
```

- [Can I use... Support tables for HTML5, CSS3, etc](https://caniuse.com/?search=filter)
