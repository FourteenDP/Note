# 弱类型、强类型、动态类型、静态类型语言

![[Pasted image 20230310011911.png]]

## 弱类型

- 变量的类型可以随意改变
- 语言
  - JavaScript
  - TypeScript 因为是 JavaScript 的超集，所以也是弱类型语言

## 强类型

- 变量的类型不可以随意改变

## 动态类型

- 变量的类型在运行时才能确定

## 静态类型

- 变量的类型在编译时就能确定

## 注意

强类型和弱类型是相对的，没有明确的定义。
部分语言同时具有强类型和弱类型，比如 C#。
