# Neovim 的安装

## 直接 Debian 远程仓库安装

**版本较旧**

```
apt install neovim
```

## 从 GitHub 仓库下载安装

- 在这 [Neovim Releases](https://github.com/neovim/neovim/releases) 找到对应版本下载到系统中

```shell
wget <url>
```

- 使用 `apt` 安装 `neovim`

```shell
sudo apt install ./nvim-linux64.deb
```
