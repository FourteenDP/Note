## 什么是 BEM 命名规范

BEM 是 Block Element Modifier 的缩写,为什么要使用 BEM 命名规范呢?

- Block: 一个独立的组件
- Element: 一个组件的一部分
- Modifier: 一个组件的状态

```css
.block {}
.block__element {}
.block--modifier {}
```

## BEM 命名规范的优点

- 有利于 CSS 的复用
- 有利于 CSS 的维护
- 有利于 CSS 的扩展
