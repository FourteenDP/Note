# 浏览器 URL 字符串长度限制

HTTP 协议的 RFC 规范中并没有详细的规定 URL 最大字符串长度，然而在浏览器中或服务器中总会存在限制

一般限制为**2046 个字符串**根据系统浏览器服务器各有所异
